# Dozin Filmes Grátis

Este é um site de filmes grátis, usando apenas conteúdos de domínio público da Internet Archive.  
O site permite:

- Buscar filmes
- Assistir filmes no player embutido
- Navegar em lista de filmes grátis

## Como usar

1. Abra o arquivo `index.html` no navegador, ou acesse o site hospedado.  
2. Clique em um filme para assistir.  
3. Use a barra de busca para filtrar por título.

## Hospedagem gratuita sugerida

- GitHub Pages: https://pages.github.com  
- Neocities: https://neocities.org
